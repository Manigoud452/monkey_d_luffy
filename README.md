One Piece Gear 5 Reveal

An interactive, animated landing page inspired by One Piece: Gear 5. This project uses HTML5 Canvas, CSS animations, and JavaScript to create an immersive split-screen reveal where users can uncover Luffy's Gear 5 transformation through mouse movement.

✨ Features
🎨 Interactive cursor-based reveal animation
⚡ Smooth HTML5 Canvas effects
🌟 Modern responsive UI design
🎭 Animated typography and stylish layout
📱 Mobile-friendly interface
🚀 Pure HTML, CSS, and JavaScript (No frameworks)
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
HTML5 Canvas API
📌 Project Overview

This project recreates the excitement of Monkey D. Luffy's Gear 5 awakening using modern front-end web technologies. As users move their cursor across the screen, hidden visuals are revealed through dynamic canvas masking and animation effects, delivering an engaging interactive experience.
